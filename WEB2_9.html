Hypertext Markup Language(HTML)is the standard markup language for creating web pages and web applications. Web browsers recieve HTML documents from a web server or from a local storage and render them into multimedia web pages. HTML describes the structure of a web page sementically and originally included cues for the appearance of document.

HTML elements are thr building blocks of HTML pages. With HTML constructs, images and othe objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structed documents by denoting structual semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delined by tags, written using angle brackets.
